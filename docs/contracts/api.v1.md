# API-контракт сервиса

## Эндпоинты
- `GET /api/v1/top?n=10`
  Возвращает текущий список самых частых запросов.
- `GET /api/v1/stop-words` (админ)
  Возвращает текущий список стоп-слов.
- `POST /api/v1/stop-words` (админ)
  Добавляет слово в стоп-лист.
- `DELETE /api/v1/stop-words/{word}` (админ)
  Удаляет слово из стоп-листа.
- `GET /api/v1/health`
  Проверка, что сервис запущен и готов отвечать.
- `GET /metrics`
  Ручка с метриками для Prometheus.
- `GET /swagger/`
  Страница Swagger для просмотра и проверки API.
- `GET /openapi.yaml`
  OpenAPI-файл, который использует Swagger.

## Коды ответов
- `200` / `201` - запрос выполнен успешно.
- `400` - неверные параметры или тело запроса.
- `401` - отсутствует или неверный `X-Admin-Token`.
- `404` - слово не найдено при удалении из стоп-листа.
- `409` - слово уже есть в стоп-листе.
- `503` - данные временно недоступны.
